Status: **STAFF-ONLY / INTERNAL-ONLY.**

The in-app guide is available only to staff and superusers at
`/staff/setup-guide/` (route name `staff_setup_guide`). It selects one source
file according to the user's current language:

- English: [`STAFF_SETUP_GUIDE.en.md`](STAFF_SETUP_GUIDE.en.md)
- 中文: [`STAFF_SETUP_GUIDE.zh.md`](STAFF_SETUP_GUIDE.zh.md)

There is no ordinary-member or public help surface. These practical church-staff
guides describe current shipped behavior only and make no production-readiness
claim. The guides and their in-app rendering add no model, migration, or
database write.
